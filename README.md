# 🌊 MALDIVES: The Pearl of Indian Ocean

This repository contains a presentation about the **Maldives**, focusing on its **history, geographical location, culture, festivals, cuisine, and major tourist attractions**.

The project was created as part of the **Online Spoken English and Phonetics course** at **MENTORS'**.

---

## 📑 Presentation Overview

The presentation explores several important aspects of the Maldives including:

- 📜 History
- 🌍 Location & Info
- 🗣️ Language
- 🎭 Culture
- 🎉 Festival and Celebration
- 🍽️ Food
- 🏝️ Tourist Attractions

It highlights the Maldives as a unique island nation known for its **natural beauty, rich culture, and global tourism importance**.
